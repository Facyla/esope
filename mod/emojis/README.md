# Emoji Detection

Adds Unicode Emojis in Elgg. 

## Description
- Detects emojis
- Encodes emojis as HTML entity for MySQL storage (&#xXXXXX;)
- Converts text :emojis: to HTML or Unicode emojis
- Provide endpoint for emojis search
